# Deep-Learning-TTS-Project
# 🎙️ AI Text-to-Speech Generator

A Deep Learning project that converts text into natural speech using SpeechT5.

---

## 🚀 Features

- Convert text to speech
- Interactive web interface using Gradio
- Neutral and consistent voice output
- Real-time audio generation

---

## 🧠 Technologies Used

- Python
- PyTorch
- Hugging Face Transformers
- SpeechT5 Model
- Gradio
